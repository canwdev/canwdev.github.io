    з¬¬дёҖж¬ЎиҝҗиЎҢпјҢжІЎжңүubuntuй•ңеғҸпјҢе°Ҷдјҡд»Һ DockerHub дёӢиҪҪпјҲеҰӮжһңдёӢиҪҪеӨұиҙҘиҜ·зңӢ #Docker Hub й•ңеғҸй…ҚзҪ®пјүпјҢе®ҢжҲҗеҗҺдјҡиҮӘеҠЁ йҷ„зқҖ еҲ°иҜҘе®№еҷЁйҮҢгҖӮ

    еңЁе®№еҷЁзҺҜеўғйҮҢпјҢиҝҗиЎҢ ps -aux еҸҜд»ҘжҹҘзңӢе®№еҷЁе®№еҷЁдёӯиҝҗиЎҢзҡ„е…ЁйғЁиҝӣзЁӢпјҢеҪ“ /bin/bash йҖҖеҮәд№ӢеҗҺпјҢе®№еҷЁд№ҹе°ұеҒңжӯўиҝҗиЎҢгҖӮ

    дҪҝз”ЁиҝҷжқЎе‘Ҫд»ӨеҸҜд»Ҙйҷ„зқҖеҲ°иҝҗиЎҢдёӯзҡ„е®№еҷЁдёҠпјҡdocker attach bob_the_container

    е®№еҷЁе‘ҪеҗҚ¶

    еҢ…еҗ«д»ҘдёӢеӯ—з¬Ұпјҡе°ҸеҶҷеӯ—жҜҚaгҖҒzгҖҒеӨ§еҶҷеӯ—жҜҚA-zгҖҒж•°еӯ—0гҖҒ9гҖҒдёӢеҲ’зәҝгҖҒеңҶзӮ№гҖҒжЁӘзәҝпјҲеҰӮжһңз”ЁжӯЈеҲҷиЎЁиҫҫејҸжқҘиЎЁзӨәиҝҷдәӣз¬ҰеҸ·пјҢе°ұжҳҜ [a-zA-Z0-9_.-]пјҢеҸҜд»Ҙз”Ёе®№еҷЁе‘ҪеҗҚд»Јжӣҝе®№еҷЁidпјҢеӣ жӯӨе®№еҷЁе‘ҪеҗҚеҝ…йЎ»жҳҜе”ҜдёҖзҡ„

    • еҗҜеҠЁж–°зҡ„ж—¶е‘ҪеҗҚпјҡdocker --name bob_the_container run -i -t ubuntu /bin/bash

    • йҮҚе‘ҪеҗҚпјҡdocker rename 3d0746cf585e bob_the_container

    • жҹҘзңӢжңҖеҗҺиҝҗиЎҢзҡ„е®№еҷЁпјҡdocker ps -l

    • еҲ йҷӨе®№еҷЁпјҡdocker rm 3d0746cf585e

    • еҗҜеҠЁе®№еҷЁпјҡdocker start bob_the_containerпјҲйҮҚеҗҜеҲҷжҳҜ restartпјү

    еҲӣе»әе®ҲжҠӨејҸе®№еҷЁ¶

    дёҠйқўжүҖеҲӣе»әзҡ„е®№еҷЁйғҪжҳҜдәӨдә’ејҸиҝҗиЎҢзҡ„е®№еҷЁпјҢд№ҹеҸҜд»ҘеҲӣе»әй•ҝжңҹиҝҗиЎҢзҡ„е®№еҷЁпјҢеҸ«еҒҡе®ҲжҠӨејҸе®№еҷЁпјҢжІЎжңүдәӨдә’ејҸдјҡиҜқпјҢйқһеёёйҖӮеҗҲиҝҗиЎҢеә”з”ЁзЁӢеәҸе’ҢжңҚеҠЎгҖӮ

    docker run --name daemon_dave -d ubuntu /bin/sh -c "while true; do echo hello world; sleep 1; done"
