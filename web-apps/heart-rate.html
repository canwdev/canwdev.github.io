使用限制与配置提醒：

  • 广播开关：请务必先开启手表的【心率推送】。
  • 浏览器环境：必须使用基于 Chromium 内核的浏览器。Safari 和 Firefox 默认不支持 Web Bluetooth。
  • 安全限制：Web Bluetooth 要求必须在安全上下文（HTTPS）中运行，或者直接在本地双击打开（即 file:/// 协议的本地 html，或者 localhost）。
  • 数据导出：连接设备后所有心率读数会自动记录，可随时点击「下载心率数据」导出完整 JSON。
  • 画中画：点击「画中画」将实时心率与趋势图浮窗显示（需 Chrome 116+ / Edge 116+）。
