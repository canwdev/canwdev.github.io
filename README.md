## 个人博客

本仓库使用了mkdocs模板，主要的 Obsidian 文档在 `docs` 目录内。

本地运行需要 python3 环境

```shell
# 安装依赖
pip install mkdocs
pip install mkdocs-material
pip install mkdocs-roamlinks-plugin

# 运行
mkdocs serve
```
