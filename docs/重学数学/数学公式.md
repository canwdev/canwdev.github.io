## 导数

## 导数运算法则

| $[u(x)\pm v(x)]'=u'(x)\pm v'(x)$ |
| -------------------------------- |
|                                  |
|                                  |
|                                  |
|                                  |

## 求导公式

| 公式                             | 示例                                                         |
| -------------------------------- | ------------------------------------------------------------ |
| $(C)'=0$                         | $(5)'=0$ , $(10)'=0$                                         |
| $(x^a)'=ax^{a-1}$                | $(x^3)'=3x^2$, $(x^5)'=5x^4$                                 |
| $(a^x)=a^xln(a)$                 | $(2^x)=2^xln(2)$, $(7^x)'=7^xln(7)$                          |
| $(e^x)'=e^x$                     | $(e^x)'=e^x$                                                 |
| $log_a(x)'=\frac{1}{xln(a)}$     | $log_{10}(x)'=\frac{1}{xln(10)}$, $log_{6}(x)'=\frac{1}{xln(6)}$ |
| $[ln(x)]'=\frac{1}{x}$           |                                                              |
| $[sin(x)]'=cos(x)$               |                                                              |
| $[cos(x)]'=-sin(x)$              |                                                              |
| $[tan(x)]'=\frac{1}{cos^{2}{x}}$ |                                                              |
| $[arctan(x)]'=\frac{1}{1+x^2}$   |                                                              |
