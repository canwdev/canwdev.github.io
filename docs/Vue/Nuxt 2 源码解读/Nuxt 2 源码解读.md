## 前置步骤

1. 下载 [v2.18.1](https://github.com/nuxt/nuxt/releases/tag/v2.18.1) 源码
2. cd 到源码目录
3. 执行 `yarn` 安装依赖
4. 执行 `yarn build` 构建
5. 运行示例：`yarn nuxt examples/hello-world/` 

> 请查看此文件夹下的 draw.io 画布！
